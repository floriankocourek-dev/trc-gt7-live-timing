## Installation auf einem Android-Smartphone

1. Die Datei `TRC-GT7-Collector-Android-v0.1.0.apk` auf das Smartphone schicken, z. B. per Discord.
2. Auf dem Smartphone die APK antippen.
3. Falls Android blockiert: Installation aus unbekannten Quellen fuer Discord/Dateimanager erlauben.
4. Installation bestaetigen.
5. App `TRC GT7 Collector` starten.

## Nutzung

1. Smartphone und PlayStation muessen im selben WLAN/LAN sein.
2. GT7 muss laufen und in einer Session sein, in der Telemetrie gesendet wird.
3. In Race Control das Rennen und den Entry anlegen.
4. In der Android-App:
   - Server stehen lassen: `https://trc-gt7-live-timing.onrender.com`
   - Race Code eingeben
   - `Connect` druecken
   - Team/Car auswaehlen
   - Current Driver auswaehlen
   - Team PIN eingeben
   - PlayStation IP eintragen
   - `Start Sending` druecken

## Wichtig fuer Freunde

Android zeigt beim ersten Installieren eine Sicherheitswarnung, weil die App nicht aus dem Play Store kommt. Fuer den internen Test ist das normal.

Die App braucht nur Netzwerkrechte. Sie greift nicht auf Mikrofon, Kamera, Bildschirmaufnahme, Tastatur, Browserdaten oder persoenliche Dateien zu.
